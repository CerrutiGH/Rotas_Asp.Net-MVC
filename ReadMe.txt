Para fazer a execução do código, siga os passos abaixo.

> Execute o script sql "NoticiasDB.sql" no SGBD MySQL Workbench.

> Na pasta "Exercício_GabrielCerruti", abra a pasta "Connection".

> Na pasta "Connection", abra o "Conexao.cs" e edite o MySqlConnection, colocando o seu server, seu user e sua senha do banco.

> Agora é só executar. :)
